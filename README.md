CyberShield is a computer security software program.

It allows you to analyze binaries, URLs, and IP addresses.

It can also generate analysis reports in PDF format using **Groq** and **FPDF2**.

In addition, it can function as a proxy server using **MITMProxy**.

To use it, please download and install the dependencies : **pip install -r requirements.txt**.

Once that’s done, run the script : **main.pyw**
